🐾 MASCOTAS INTERACTIVAS DE PIXEL ART

Una aplicación de escritorio desarrollada en Python + PySide6 que simula mascotas virtuales con estilo pixel art, animaciones e interacción básica (base hecha casi en su totalidad con IA).

✨ CARACTERISTICAS DE LA APP

🖥️ Interfaz gráfica con PySide6
🎮 Sistema base de mascota interactiva
🎨 Estilo pixel art
⚡ Estructura modular para escalabilidad
🧪 Entorno virtual para desarrollo aislado

📦 TECNOLOGIAS UTILIZADAS

-Python 3.14.5
-PySide6
-shiboken6

📄 requirements.txt

Este proyecto utiliza:

PySide6==6.11.1
PySide6_Addons==6.11.1
PySide6_Essentials==6.11.1
shiboken6==6.11.1

CAMBIOS RECIENTES DEL PROYECTO:

🆕 Estructura inicial del proyecto creada
🖼️ Implementación base de interfaz con PySide6
🐾 Sistema inicial de mascota interactiva pudiendo caminar de forma aleatoria y mover con el mouse
⚙️ Configuración de entorno virtual (venv)

INSTALACION DEL PROYECTO:

1. CLONAR EL REPOSITORIO:
-------------------------------------------------------------------------------
| git clone https://github.com/Donxoripan/mascotas-interactivas-pixel-art.git |
| cd mascotas-interactivas-pixel-art                                          |
-------------------------------------------------------------------------------

2. CREAR ENTORNO VIRTUAL EN WINDOWS (PowerShell): 
------------------------
| python -m venv venv  |
------------------------

2.1 Activar:
-------------------------
| venv\Scripts\activate |
-------------------------

2.5 EN EL CASO DE USAR SISTEMA OPERATIVO LINUX / MAC:
----------------------------
| python3 -m venv venv     |
| source venv/bin/activate |
----------------------------

3. INSTALAR DEPENDENCIAS
-----------------------------------
| pip install -r requirements.txt |
-----------------------------------

4. EJECUTAR LA APLICACION:
------------------
| python main.py |
------------------
